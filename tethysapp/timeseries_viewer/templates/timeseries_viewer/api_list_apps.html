list_apps API method documentation

Shows all HydroApps that can be used by CUAHSI HydroClient

Input

No input parameters

Output

Output in JSON format:

name
Name of the HydroApp
description
Description of the HydroApp (how is it used)
min_series
Minimum required number of time series
max_series
Maximum allowed number of time series
url
URL for launching the HydroApp
icon
URL of the HydroApp's icon

Call list_apps Method:

http://apps.hydroshare.org/apps/timeseries-viewer/api/list_apps/ {% endblock %}
